# Course Registration System

A Python-based Course Registration System using **OOP**, **Service Layer**, and **Tkinter GUI**.

## Features
- User Authentication (Admin / Lecturer / Student)
- Course Registration by Semester
- Learning History & Academic Records
- Admin Course & User Management
- Notification System

## Technologies
- Python 3
- Tkinter (GUI)
- JSON (Data Storage)
- OOP & MVC-like Architecture
